- [[HTML]]
	- \<!DOCTYPE html> 声明
	- 最小 HTML5 文档
		- ```
		  <!DOCTYPE html>
		  <html>
		  <head>
		  <meta charset="utf-8">
		  <title>文档标题</title>
		  </head>
		   
		  <body>
		  文档内容......
		  </body>
		   
		  </html>
		  ```
		- meta charset="utf-8" 保证可以输入中文
		- HTML 标签
			- HTML 标记标签通常被称为 HTML 标签 (HTML tag)。
			- HTML 标签是由*尖括号*包围的关键词，比如 <html>
			- HTML 标签通常是*成对出现*的，比如 \<b> 和 </b>
			- 标签对中的第一个标签是*开始标签*，第二个标签是*结束标签*
			- 开始和结束标签也被称为*开放标签*和*闭合标签*
